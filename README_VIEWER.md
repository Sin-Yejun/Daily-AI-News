# Daily News Viewer

Markdown 파일을 쉽게 보기 위한 로컬 웹 뷰어입니다.

## 실행 방법

1. 의존성 설치 (최초 1회)
   ```bash
   npm install
   ```
2. 서버 실행
   ```bash
   node server.js
   ```
3. 브라우저 접속
   - [http://localhost:3000](http://localhost:3000)

## 기능

- **폴더 자동 탐지**: `오디오` 폴더 등을 제외한 컨텐츠 폴더를 자동으로 불러옵니다.
- **최신순 정렬**: 파일 이름(날짜)을 기준으로 내림차순 정렬됩니다.
- **다크 모드 UI**: 눈이 편안한 모던한 다크 테마를 적용했습니다.
- **이미지 지원**: Markdown 내부의 상대 경로 이미지를 올바르게 표시합니다.

## 주의사항

- `git pull`로 새로운 데이터가 들어오면, 브라우저를 새로고침하면 반영됩니다.
- 새로운 폴더가 추가되면 자동으로 리스트에 표시됩니다 (단, excluded 목록에 포함되지 않은 경우).
