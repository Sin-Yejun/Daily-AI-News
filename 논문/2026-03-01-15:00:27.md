요청해주신 논문(총 10편)에 대해 전문적 요약(논문 구조 기반)과 중학생용 쉬운설명 두 가지를 모두 제공해 드리겠습니다. 다만 현재 제 환경에서는 외부 웹 페이지나 PDF를 직접 열람할 수 없습니다. 그래서 정확하고 논문 전체 기준의 상세 분석을 제공하려면 다음 중 하나를 선택해 주세요.

옵션 A — PDF 업로드(권장)
- 각 논문의 PDF 파일을 이 채팅에 업로드해 주세요. 그러면 전체 본문을 전부 분석하여 요청하신 형식(표지 썸네일 포함 템플릿, 전문 요약 + 쉬운설명, 관련분야, 활용·시사점 등)으로 논문별로 자세히 정리해 드립니다.

옵션 B — 초록(abstract) 또는 목차 제공
- 각 논문의 초록 또는 목차(또는 주요 섹션 헤더)를 붙여 넣어 주세요. 전체 본문보다는 덜 상세하지만, 주어진 정보로 구조에 맞춘 요약과 쉬운설명을 작성해 드립니다.

옵션 C — 링크 허용(제가 직접 접근 불가)
- 제공하신 Hugging Face 링크들을 제가 바로 열람할 수는 없습니다. 만약 사용자가 링크의 PDF를 직접 다운로드한 뒤 업로드해 주면 진행 가능합니다.

추가 확인 사항
1. 모든 논문을 한 번에(10편 전부) 받을까요, 아니면 우선순위(예: 상위 3편)를 정할까요?  
2. 요약 분량에 선호가 있으신가요? (예: 전문 요약 — 논문 길이 기준 800–1,200단어 / 간단 요약 — 150–250단어)  
3. 코드/재현 관련 코드 블록이나 실험 재현 가이드를 포함하길 원하십니까?

원하시는 옵션을 알려주시면 곧바로 분석을 시작하겠습니다.
